# Приют для животных - Desktop приложение

WPF приложение для управления приютом для животных, разработанное по стандартам WorldSkills.

## Технологии

- C# .NET Framework 4.8
- WPF (Windows Presentation Foundation) - классический формат проекта
- PostgreSQL
- Npgsql 4.1.14
- BCrypt.Net-Next 4.0.3

## Формат проекта

Проект использует **классический формат WPF App (.NET Framework)**:
- Файл проекта: `ShelterApp.csproj` (старый формат, не SDK-style)
- NuGet пакеты: `packages.config`
- Конфигурация: `App.config`
- Метаданные сборки: `Properties/AssemblyInfo.cs`

## Структура проекта

```
ShelterApp/
├── App.xaml - Глобальные стили и ресурсы
├── Windows/ - Окна приложения
│   ├── MainWindow.xaml - Главное окно с навигацией
│   ├── LoginWindow.xaml - Окно входа
│   └── RegisterWindow.xaml - Окно регистрации
├── Pages/ - Страницы приложения
│   ├── CatalogPage.xaml - Каталог животных с фильтрами
│   ├── AnimalDetailPage.xaml - Детальная информация о животном
│   ├── FavoritesPage.xaml - Избранные животные
│   ├── ProfilePage.xaml - Профиль пользователя
│   └── AdminPage.xaml - Панель администратора
├── Models/ - Модели данных
├── Repositories/ - Репозитории для работы с БД
├── Services/ - Сервисы (Auth, Session)
├── Helpers/ - Вспомогательные классы (Manager)
└── Database/ - SQL скрипты
```

## Установка и запуск

### Требования

- Windows 10/11
- .NET Framework 4.8
- PostgreSQL 12+
- Visual Studio 2019/2022

### Настройка базы данных

1. Установите PostgreSQL
2. Создайте базу данных:
```sql
CREATE DATABASE shelter_db;
```

3. Выполните скрипты из папки `Database/`:
```bash
psql -U postgres -d shelter_db -f Database/create_tables.sql
psql -U postgres -d shelter_db -f Database/insert_test_data.sql
```

4. Настройте строку подключения в `Database/DatabaseConnection.cs`:
```csharp
private static string connectionString = "Host=localhost;Port=5432;Database=shelter_db;Username=postgres;Password=ваш_пароль";
```

### Запуск приложения

1. Откройте `ShelterApp.csproj` в Visual Studio
2. Восстановите NuGet пакеты
3. Соберите проект (Build → Build Solution)
4. Запустите (F5)

## Функциональность

### Для всех пользователей:
- Просмотр каталога животных
- Фильтрация по типу, полу, размеру
- Просмотр детальной информации о животном
- Подача заявки на опекунство

### Для авторизованных пользователей:
- Добавление в избранное
- Управление профилем
- Просмотр статуса заявок

### Для администраторов:
- Управление животными (добавление, редактирование, удаление)
- Управление заявками (одобрение, отклонение)
- Просмотр всех данных

## Архитектурные особенности

Проект разработан согласно стандартам WorldSkills:

1. **Навигация** - использование Frame для переключения между страницами
2. **Глобальные стили** - определены в App.xaml для единообразия дизайна
3. **Менеджер навигации** - статический класс Manager для управления Frame
4. **Репозитории** - паттерн Repository для работы с данными
5. **Сервисы** - бизнес-логика вынесена в отдельные сервисы
6. **Безопасность** - хеширование паролей с помощью BCrypt
